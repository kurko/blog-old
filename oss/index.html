Open-source projects

RUBY GEMS & JS LIBS

Here are some projects I created.

Vito (github)

A promising provisioning gem to replace Chef and Puppet. You define a Gemfile-like file with what you want to be installed in the server. The rest is taken care of by Vito. A must see project.

Crane (github)

This gem will analyze what are the latest modified files and upload them to a FTP server. Say you want to upload only file modified within 2 hours? This gem does that.

Omniscient (github)

Simple gem for importing remote databases via SSH.

Flying (github)

Although the gem is out-dated, the premise is very good. Basically, you define a Gemfile-like file with the external dependencies your app has, such as Fedex, AdWords or others. Whenever one of those websites is down, you're notified.

Richer (github)

Super light-weight Javascript lib to have a simple WYSIWYG textarea editor. Has basic features, such as bold and italic.

RESEARCH, UNDOCUMENTED GEMS

Kanji (github)

This is a gem for recognizing japanese Kanjis in images with artificial neural nets. Pretty neat. You can see it working in the test files. I basically teach the neural nets the meaning of each kanji and then it becomes capable of recognizing it later.

Emerald.js (github)

Yet another Javascript MVC framework. Although I'll not use it because we have Ember.js, you can check the source code to see how to build controllers, ORM, auto-update of view elements, and others.

Vim plugins

smartest.vim (github)

With this plugin, whenever you run `RunTestFile()` smartest.vim will check if you're in a Ruby or JS project, if you're using spring, zeus or other gem to make your specs fast, and run specs on the fastest mechanism possible. I can't live without this plugin anymore.

autocoder.vim (github)

One of the biggest barrier when coding is the IDE you use. This plugin will generate Ruby classes and specs for you. No more excuses like "creating yet another class is painful".
